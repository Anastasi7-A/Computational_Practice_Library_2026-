# Веб-приложение для управления библиотечным фондом факультета с учётом выдачи книг, системой штрафов и аналитикой.

## Проект представляет собой полноценную информационную систему онлайн-библиотеки, разработанную в рамках учебной практики по базам данных и прикладной разработке. Приложение автоматизирует основные библиотечные процессы: учёт книг, выдачу и возврат читателям, расчёт штрафов за просрочку, а также предоставляет аналитику по популярности изданий.

В проекте используются : Python +3.10; Flask; MySQL; HTML5.

**Обязательные возможности:**
* Каталог книг — хранение информации о названии, авторе, годе издания, жанре и количестве экземпляров
* Учёт читателей — регистрация студентов и преподавателей
* Выдача и возврат — фиксация операций с книгами с привязкой к читателям
* Система штрафов — автоматический расчёт штрафов за просрочку возврата по библиотечным правилам
* Поиск и фильтрация — по названию, автору, жанру, году издания
* Полный CRUD — создание, чтение, обновление и удаление записей через веб-интерфейс

**Дополнительные возможности:**
🖼️ Обложки книг — загрузка и отображение изображений обложек
🔒 Бронирование — возможность забронировать книгу, если все экземпляры заняты
📊 Визуализация — графики популярности книг и активности читателей
Удалённое развёртывание — возможность деплоя на внешний сервер
